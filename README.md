# Media Player
Тестовое задание для Intech

## Функионал:
* Поиск треков из iTunes
* Проигрывание выгранного трека

## Стек
* Kotlin
* MVVM
* Jetpack Navigation
* ExoPlayer
* RxJava2
* OkHttp, Retrofit
